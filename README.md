# 🚀 AI Dev Assistant

一个基于 RAG（检索增强生成）的本地 AI 开发助手，用于提升代码理解、Bug 定位与开发效率。

---

## ✨ 项目简介

在中大型代码项目中，开发者往往需要花费大量时间理解代码逻辑或定位问题。本项目通过引入大模型能力，结合本地代码语义检索，实现“快速理解 + 智能问答”。

---

## 🔑 核心功能

- 🔍 **代码语义检索**
  - 将本地代码切分并向量化
  - 基于相似度检索相关上下文

- 🤖 **AI 问答**
  - 函数解释
  - Bug 定位
  - 逻辑分析

- 🧠 **多 Agent 架构**
  - Retriever：负责上下文检索
  - Generator：负责答案生成

- 🖥️ **Qt 桌面界面**
  - 本地运行
  - 支持异步请求（避免 UI 卡顿）

---

## 🏗️ 系统架构

![architecture](docs/architecture.png)
